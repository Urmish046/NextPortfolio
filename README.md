# 🚀 Personal Developer Portfolio

A modern, highly optimized personal portfolio built to showcase my projects, skills, and contact information. Developed with a focus on clean architecture, smooth animations, and fast media delivery.

**🔗 [Live Demo - View Here](https://urmish-portfolio.vercel.app/)**

---

## ✨ Key Features & Real-World Logic

* **Serverless Contact Form:** Built a custom Next.js API route using `Nodemailer` and SMTP to securely handle form submissions without exposing credentials.
* **Media Optimization:** Integrated **Cloudinary** (CDN) to serve high-quality project showcase videos instantly, avoiding heavy local storage and buffering.
* **State-Based Rendering:** Implemented dynamic "View All" functionality using React state for efficient DOM updates without page reloads.
* **Smooth Animations:** Utilized `framer-motion` for declarative, scroll-triggered, and interactive UI animations.
* **Fully Responsive:** Designed with Tailwind CSS to ensure a pixel-perfect experience across all mobile, tablet, and desktop devices.

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Media Delivery:** Cloudinary
* **Backend Email:** Nodemailer (SMTP API)
* **Deployment:** Vercel
